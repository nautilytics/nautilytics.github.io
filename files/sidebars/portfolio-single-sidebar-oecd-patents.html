OECD Patent Visualization

Context
USDA/NIFA wanted an innovative way to visualize the trends in global biotechnology patents over an 11 year period. I determined this project would be perfect for combining the data visualization power of d3.js with the 3D capabilities of three.js.
Data
International data on patent applications from 2000 to 2011 from the OECD Patent Database.
Tech Notes
  • The 3D elements in the application were designed using 3ds Max and Blender.
  • The front-end was built using three.js, d3.js, jQuery, and jQuery UI.
