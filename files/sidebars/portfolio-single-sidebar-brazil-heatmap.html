Brazil Grain Storage Facility Heatmap

Context
During my research on the location of grain storage facilities in Brazil, I found the current resources to be inadequate. There was a single data repository, but multiple steps were required to see just one of the 18,000+ facilities displayed on a map. Wanting to view multiple facilities at once, I collected the geocoded data and ran it through the Google Maps heatmap library to show the areas of Brazil where grain storage facilities are abundant and where they are lacking.
Data
Over 18,000 geocoded grain storage facilities with many columns of qualitative data from CONAB.
Tech Notes
  • Data collection and manipulation were done using Python libraries such as BeautifulSoup and NumPy.
  • Visualization was done using the Google Maps heatmap library.
